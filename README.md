# esp8266-s88n-lib
S88n library for ESP8266

### S88n pin configuration
| ESP8266 | Variable       | Description |
|---------|----------------|-------------|
| GPIO15  | `S88DataPin`   | Data in     |
| GPIO13  | `S88ClkPin`    | Clock       |
| GPIO12  | `S88PSPin`     | PS/LOAD     |
| GPIO14  | `S88ResetPin`  | Reset       |
